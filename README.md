# pythonAssignment_palindrome
## 回文数判断。设n是一个任意自然数，如果n的各位数字反向排列所得自然数与n相等，则n被称为回文数。从键盘输入一个5位数字，请编写程序决断这个数字是不是回文数。
### 注意：1、对非5位或非纯数字输入，必须输出错误提示 2、结果输出必须是精确的：是回文数 或 不是回文数 3、推荐使用第三章学习的字符串操作函数
